IFA


Introduction

IFA is a library for Interative Flow Analysis of computer programs.

This software is for compiler developers and program analysis researchers. It is based on work initially done by the author at the University of Illinois Champaign-Urbana. Papers describing this work can be found at papers. This implementation was begun as part of the Chapel programming language.

These pages are under construction. For more information, please contact the author directly.

Features
  • Iterative Flow Analysis
  • Time/Space Efficient
  • Comes with full source
  • Written in C++
  • Open Source Licence (MIT)
